# Quickstart: 2.5D Visual & Audio Presentation

**Feature**: `002-2-5d-visual-audio`

## Prerequisites

- Godot 4.6+ CLI (see root `README.md`)
- GUT 9.4 addon in `godot/addons/gut/`

## Development

```bash
# Open editor
godot --path godot

# Run game (table scene for presentation work)
godot --path godot
# Press F5 or use Boot → Home → Setup → Table
```

## Testing (TDD — write tests before implementation)


## Asset workflow

### Regenerate procedural assets

```bash
python godot/assets/audio/generate_audio.py   # BGM + 15 SFX (numpy + ffmpeg)
python godot/assets/models/generate_models.py  # faceted glTF dogs/props
godot --headless --path godot --import
```

### 3D models

`generate_models.py` emits faceted glTF under `godot/assets/models/` (chibi dogs in hoodies,
dealer dog, round table, chip, shoe, lamp, plant). `table_3d.gd` loads these at runtime.
**Deferred (T117)**: swap for licensed final glTF while preserving Ref A faceted matte style.

### 2D theme

Edit `godot/assets/themes/mix2_shell.tres` in Godot Theme editor. Apply via `ui_theme.gd`.

### Audio

Procedural Ogg files are generated by `godot/assets/audio/generate_audio.py` (Animal Crossing /
Stardew-inspired: soft marimba BGM, gentle card/chip/UI SFX). Replace with licensed final mixes in
a DAW when ready; keep filenames per `contracts/audio-contract.md`.

## Manual verification checklist

1. Home → Setup → Table: sidebar and menus share Ref B green/brown panels and rounded buttons.
2. Table: 3D cards on felt per seat; shoe shows remaining count.
3. Narrow window (< 900 px): sidebar stacks above table; counts visible without scroll.
4. Tutorial lesson: recommended action shows ★ prefix, gold border, and tooltip (not color-only).
5. Only legal actions appear near learner hand; Tab focuses visible buttons (44 pt min height).
6. Analytics button opens overlay charts; closing returns to play.
7. BGM loops at table; stops on Home; SFX on each action category.
8. Enable reduced motion: deal/chip/dog/outcome animations snap instantly; highlight stays static.
9. Bet confirm: chip bounces at learner seat; deal triggers dog reaction; hand settle shows win sparkle or loss dim cue.
10. Sidebar Analytics/Options buttons meet 44 pt touch target and accept keyboard focus.

## Build exports

Presentation assets ship with standard exports:

```bash
powershell -ExecutionPolicy Bypass -File scripts/build-steam.ps1
powershell -ExecutionPolicy Bypass -File scripts/build-web.ps1
```

### Web export smoke (manual — requires Godot editor or CLI with export templates)

Web export cannot be validated headlessly without Godot installed and export presets configured.

1. Install Godot 4.6+ with Web export templates (Editor → Manage Export Templates).
2. Run `powershell -ExecutionPolicy Bypass -File scripts/build-web.ps1` from repo root.
3. Serve the `dist/web/` folder locally (`npx serve dist/web` or similar).
4. Open in browser; click anywhere on canvas to unlock audio autoplay.
5. Navigate Home → Setup → Table; confirm BGM starts after first input.
6. Toggle music/SFX in Options; confirm independent mute paths.
7. Enable reduced motion in Options; confirm animations snap on next hand.
